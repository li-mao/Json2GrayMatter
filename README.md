# Json2GrayMatter
gray-matter 将md文件转成json.
Json2GrayMatter 将json转成md文件.

示例

```
limao@LiMao:~/github/Json2GrayMatter (main%=) % node example.js
---
companyIntro : 
        title : 公司介绍
        content : 
                -       xxxxx，注册资本5000万。
                -       公司专注化、国际化新型品牌服务体系。
---
limao@LiMao:~/github/Json2GrayMatter (main*%=) %  
```
